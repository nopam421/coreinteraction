Trek Anywhere is an application that focuses on making one's workout on the Elliptical machine at the gym more fun, effortless and satisfying.

By the use of Virtual Reality goggles, we aim to transport the user to a different, more beautiful setting where the user is able to visualise his/her trek as he/she excercises
